\begin{abstract}
\addchaptertocentry{\abstractname} % Add the abstract to the table of contents
%
%The Thesis Abstract is written here (and usually kept to just this page). The page is kept centered vertically so can expand into the blank space above the title too\ldots
\centering

Este trabajo de graduación describe el diseño, implementación y despliegue de un sistema de firma electrónica de documentos on-premise para la Facultad de Ciencias Exactas y Tecnología (FACET) de la Universidad Nacional de Tucumán. Ante la necesidad de centralizar y agilizar los procesos administrativos de firma, actualmente manuales y distribuidos, se seleccionó y desplegó la plataforma de código abierto (Open Source) Documenso sobre una infraestructura de virtualización basada en Proxmox VE y orquestada por Coolify. La solución garantiza la soberanía de los datos mediante el auto-alojamiento y el uso de almacenamiento de objetos compatible con S3 (MinIO). El sistema se integra con el dominio institucional de Google Workspace mediante OAuth 2.0 para la autenticación de usuarios y utiliza un servicio de correo transaccional para las notificaciones automáticas. Como resultado, se entrega una solución funcional ``llave en mano'', incluyendo manuales técnicos y de procedimientos que aseguran la trazabilidad documental y la autonomía operativa del personal de TI de la facultad.
				

\end{abstract}
